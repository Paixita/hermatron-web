# Hermatron Design System

Inspired by Linear. High-contrast, dark mode, precise, and ultra-minimal.

## Visual Identity

### Colors
- **Background (Primary)**: `#0B0C0E` (Deep space)
- **Background (Secondary)**: `#151618` (Surface)
- **Background (Tertiary)**: `#1E2023` (Elevated)
- **Accent (Primary)**: `#5E6AD2` (Linear Purple)
- **Accent (Hover)**: `#737EE0`
- **Text (Primary)**: `#F7F8F8` (Off-white)
- **Text (Secondary)**: `#8A8F98` (Muted gray)
- **Text (Tertiary)**: `#60646C` (Disabled/Hint)
- **Border**: `#2B2D31` (Subtle)
- **Success**: `#4B915F`
- **Error**: `#D85151`
- **Warning**: `#D89B51`

### Typography
- **Primary Font**: `Inter`, sans-serif
- **Code Font**: `JetBrains Mono`, monospace
- **Scale**:
  - `h1`: 2rem, 700 weight, -0.02em tracking
  - `h2`: 1.5rem, 600 weight, -0.01em tracking
  - `body`: 1rem, 400 weight, normal tracking
  - `small`: 0.875rem, 400 weight

### Components

#### Buttons
- **Primary**: Purple background, white text, 6px border-radius, subtle glow on hover.
- **Secondary**: Dark gray background (`#1E2023`), white text, 6px border-radius.
- **Ghost**: Transparent background, muted text, highlights on hover.

#### Inputs
- **Style**: Dark background (`#0B0C0E`), 1px border (`#2B2D31`), 6px border-radius.
- **Focus**: Border color changes to purple (`#5E6AD2`), subtle outer glow.

#### Cards / Panels
- **Style**: Secondary background (`#151618`), 1px border (`#2B2D31`), 8px border-radius.

### Animations
- **Transitions**: `0.2s cubic-bezier(0.4, 0, 0.2, 1)`
- **Entrance**: Subtle fade-in with 4px vertical slide.

## Principles
1. **Precision**: Everything aligned to a 4px/8px grid.
2. **Contrast**: High legibility is non-negotiable.
3. **Focus**: Use the purple accent sparingly to guide the user's eye to the primary action.
4. **Minimalism**: If an element doesn't serve a purpose, remove it.
